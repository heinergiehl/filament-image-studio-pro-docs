# Image Studio Pro

Image Studio Pro adds a full image-editing workflow to Filament:

- edit image-based designs inside your panel
- manage reusable source images
- save reusable layouts
- export rendered output back into your app

## Best fit

Use Image Studio Pro when your team needs repeatable image production inside Filament instead of a separate external design tool.

## Core concepts

- `Designs`: saved working documents
- `Source Library`: reusable original images
- `Layouts`: reusable text and overlay structure
- `Exports`: rendered output files

## Why the docs are public

The commercial plugin source repo should stay private. Public docs are still the right default because buyers need installation guidance, storage guidance, screenshots, and troubleshooting before and after purchase.
