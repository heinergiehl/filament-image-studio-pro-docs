# Editor

The editor is built for a predictable image-production workflow:

1. choose or upload a source image
2. apply a reusable layout
3. edit text, overlays, and placement
4. export the result back into the app

## Snaplines

The canvas shows guide lines as layers approach alignment. Hold `Ctrl` / `Cmd` while dragging to lock the selected layer onto nearby guides.

## Upload modes

- `Design only`: use the image in the current design only
- `Add to Source Library`: store it as a reusable shared original

## Exports

Exports stay full-size even when the editor canvas is shown at a reduced preview scale.
