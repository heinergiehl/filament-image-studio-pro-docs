# Editor

The editor is built for a predictable image-production workflow:

1. choose or upload a source image
2. apply a reusable layout
3. edit text, overlays, and placement
4. export the result back into the app

## Source changes

The editor keeps source changes in one place:

- use the media rail to upload or browse Source Library
- use the `Designs` sidebar to change the current source or reopen recent work

The sidebar no longer duplicates a second upload flow.

## Snaplines

The canvas shows guide lines as layers approach alignment. Hold `Ctrl` / `Cmd` while dragging to lock the selected layer onto nearby guides.

## Upload modes

- `Design only`: use the image in the current design only
- `Add to Source Library`: store it as a reusable shared original

If you choose `Add to Source Library`, the current canvas stays unchanged until
you explicitly pick that image from the library.

## Undo / redo

Undo and redo are available directly on the stage header so they stay close to
the canvas instead of living in a separate side panel.

## Exports

Exports stay full-size even when the editor canvas is shown at a reduced preview scale.
